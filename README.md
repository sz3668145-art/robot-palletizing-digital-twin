# robot-palletizing-digital-twin
基于AI Agent+Codex智能开发的饮料箱双机器人协同码垛数字孪生3D仿真工作站，包含ABB机械臂联动搬运、全线产线时序调度、碰撞干涉检测与工业自动化RAPID程序批量生成，智能制造柔性包装开源仿真项目
## 项目简介
本项目面向智能制造工业柔性包装产线，搭建全流程虚实一体数字孪生仿真环境。场景集成双台ABB IRB120工业机械臂、滚筒输送进料、交叉横移移栽、视觉封箱检测、成品托盘智能码垛全套三维物理工业模型，实现物料流转、机器人抓取转运、时序避障联动、全局刚体碰撞干涉闭环连续动态仿真运行。

## AI智能开发技术栈
全程基于 **GPT-Codex + Hermes 多智能体协同架构**全栈驱动开发：
- AI批量自动生成RobotStudio标准RAPID机器人最优无干涉连续堆叠运动控制代码
- 向量工艺专属长记忆数据库封装箱体通用码垛复用堆叠算法
- RPA桌面自动化时序批量工况压力遍历联合极限调试脚本编排
- 多Agent长链路任务拆解深度协同推理跨机集群产线防碰撞路径验算

## 项目价值
对比传统人工逐点示教编程，AI智能体开发模式极大缩短多机器人复杂耦合非标工业工作站开发周期，降低干涉调试试错成本。项目开源可持续拓展AGV叉车入库集群调度、机器视觉来料智能分拣全厂柔性黑灯无人智造联动二次迭代研究，适用于高校工业机器人毕业设计、虚实同步离线虚拟高端智能制造AI算法科研商业合规自研落地深度验证。
